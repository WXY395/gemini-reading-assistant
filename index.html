Chrome Extension · v3.0.3

Gemini Reading Assistant

The missing reading layer for Google Gemini — navigate, search, cite, export, and screenshot your long AI conversations with ease.

Install from Chrome Web Store View on GitHub

What it does

🧭

Sidebar Navigation

Jump to any message instantly. Filter by All / Gemini / User.

🔍

Full-text Search

CtrlShiftS searches across the whole conversation. Supports Zhuyin / CJK IME input.

📷

Screenshot ToolsNEW

CtrlShiftX opens a floating menu with region / element / long-scroll capture.

✂️

Selection Toolbar

Quote or copy highlighted text with one click.

📌

Citation Clipboard

Save key passages and reinsert them into any Gemini prompt.

💾

Conversation Export

Download as Markdown, TXT (UTF-8 BOM), or JSON — nothing leaves your device.

🗂️

Auto-save & Journal

Snapshots of your visible messages persist locally so exports stay complete.

🔒

Privacy-first

All data is stored locally. The only network call is Pro license verification.
